# User Flow

## Staff Flow

1. พนักงานเลือกลูกค้าจากหน้า POS
2. กดเพิ่มจำนวนไม้ที่ซื้อ เช่น +3 หรือ +10 เพื่อสร้างรายการรอยืนยัน
3. ถ้ากดผิด พนักงานกด `ลบ 1 ไม้` หรือ `ล้างรายการ`
4. ตรวจ preview ว่าหลังยืนยันจะได้ reward หรือไม่
5. กด `ยืนยันบันทึกยอด`
6. ระบบรวมยอดสะสมและคำนวณ reward
7. ถ้าครบ 10 ไม้ ระบบเพิ่ม reward ให้ลูกค้าทันที
8. ถ้ายืนยันผิด พนักงานกด `ยกเลิกรายการล่าสุด` เพื่อคืนยอดก่อนรายการนั้น

## Customer Flow

1. ลูกค้าดูสถานะบน iPad display
2. เมื่อลูกค้ามี reward ระบบแสดงตัวเลือกของแถม
3. ลูกค้าเลือกหมูปิ้ง ข้าวเหนียว หรือเก็บสิทธิ์ไว้ก่อน
4. ระบบบันทึกการเลือกและจำลอง LINE OA message

## Portfolio Demo Flow

1. เปิดหน้า demo
2. เลือก `คุณ A`
3. กด `+3 ไม้`
4. เห็น reward เพิ่มขึ้น 1 สิทธิ์
5. เลือกของแถม
6. เห็น mock LINE OA message
