# Overview

MooPing Loyalty เป็นระบบสะสมยอดซื้อสำหรับร้านหมูปิ้งที่ต้องการประสบการณ์หน้าร้านทันสมัยกว่าบัตรสะสมแต้มแบบกระดาษ

## Problem

ร้านหน้าร้านจำยอดสะสมลูกค้าได้ยาก โดยเฉพาะเมื่อลูกค้าซื้อหลายรอบ เช่น รอบแรก 7 ไม้ รอบต่อมา 3 ไม้ ระบบควรรวมเป็น 10 ไม้และให้ reward ทันที

## Solution

ระบบบันทึกยอดซื้อให้ลูกค้าแต่ละคน คำนวณ reward อัตโนมัติ และเปิดให้ลูกค้าเลือกของแถมเองเมื่อครบเงื่อนไข

## Portfolio Value

โปรเจกต์นี้โชว์ได้ทั้ง web development, product thinking, UX สำหรับหน้าร้าน, business logic และ LINE OA integration plan
